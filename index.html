🔒 ระบบแยกสิทธิ์ชัดเจน สมาชิกจะไม่เห็นเมนูของคณะกรรมการ

⚙️ ตั้งค่าซิงค์ข้อมูล Google Sheets สองทาง (Two-Way Sync)

วาง Web App URL ที่ได้จากการ Deploy Google Apps Script ใน Google Sheets เพื่อส่งข้อมูลจากเว็บกลับไป Google Sheets ทันที

📝 ขั้นตอนการติดตั้งง่ายๆ บน Google Sheets:
  1. เปิดไฟล์ Google Sheets ของสหกรณ์ฯ ➔ ไปที่เมนู ส่วนขยาย (Extensions) ➔ เลือก Apps Script
  2. ลบโค้ดเดิมออก แล้วนำโค้ดด้านล่างนี้ไปวางแทนที่
  3. กดปุ่ม ทำให้ใช้งานได้ (Deploy) ➔ เลือก การทำให้ใช้งานได้ใหม่ (New deployment)
  4. เลือกประเภทเป็น เว็บแอป (Web app) ➔ กำหนด ผู้มีสิทธิ์เข้าถึง (Who has access) เป็น ทุกคน (Anyone)
  5. กด ทำให้ใช้งานได้ (Deploy) คัดลอก Web App URL นำมาวางใส่ในช่องด้านบนได้เลย!
